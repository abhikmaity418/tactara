Privacy

Local-first by design.

Tactara Typing Studio stores learner profiles, settings, typing sessions, lesson progress, and key statistics locally in the app environment.

What is stored

Name, selected starting level, daily goal, lesson results, WPM, accuracy, error counts, weak-key data, and exported backups when you choose to create them.

What is not required

No account, cloud login, classroom code, or online tracking is required to use the app.

Downloads

If this site is hosted publicly, normal web server logs may record basic request information such as IP address, user agent, and download time. The app itself does not need that data.
