Support

Help with Tactara.

Windows warns before installing

Locally built apps that are not code-signed can trigger SmartScreen. Choose more info only if you trust the file source.

Progress is missing after reinstall

Use the app's Export button before reinstalling, then Import after opening the new version.

The app opens but looks blank

Close and reopen Tactara. If it still fails, install the latest Windows WebView runtime or rebuild the Electron package.

macOS installer

The project includes a macOS DMG build target. Build it on macOS with bash scripts/make-mac-icon.sh and then npm run dist:mac from the Tactara.App folder, or use the included GitHub Actions workflow.
